A2SV Companion is a browser extension that detects accepted submissions on LeetCode and Codeforces and sends relevant submission data to the A2SV backend service.

## Information We Collect
- Submission metadata from LeetCode/Codeforces pages (e.g., problem title, status, language, and solution content if present on the page).
- Extension configuration values you save locally (e.g., user settings).

## How We Use Information
- To submit accepted solutions to the A2SV backend for tracking and reporting.
- To provide extension functionality.

## Data Sharing
Data is sent only to the A2SV backend for processing: https://a2sv-companion-backend.onrender.com
We do not sell data to third parties.

## Data Storage
- Local extension data is stored using Chrome storage.
- Backend storage is managed by A2SV on its servers.

## Security
We use HTTPS to transmit data to the backend. No guarantee of absolute security is provided.

## Your Choices
You can disable or remove the extension at any time.
