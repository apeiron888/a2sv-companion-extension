# A2SV Companion Extension
A2SV Companion is a Chrome extension that captures accepted LeetCode/Codeforces solutions, submits them to the A2SV backend, and helps track progress.

## Backend
Production backend: https://a2sv-companion-backend.onrender.com

## Load in Chrome (development)
1. Open chrome://extensions
2. Enable Developer mode
3. Click “Load unpacked” and select this folder

## Chrome Web Store packaging
1. Ensure icons are in static/icons
2. Ensure screenshots are in static/screenshots
3. Zip the extension root (manifest.json at the top level)
4. Upload the ZIP to the Chrome Web Store
